# Useful Commands

> By JamesSand


这个仓库记录了我从 2025 年年初以来，生活中经历的所有事情。记录在此，为了让自己不要重蹈覆辙，也为了让后来之人不要重新踩我的坑

分级结构最多不超过两级

目前的分层结构是
- 01-Engineer：工程工具相关，包括 Git、Linux 命令、Python、Conda、Jupyter、tmux 等开发环境配置
- 02-LLM-RL：大语言模型与强化学习，包括 LLM 训练、并行化策略、量化技术、Flash Attention、TRL、VERL 等
- 03-LLM-Agent：AI Agent 工具和使用，包括 Claude、MCP、Ollama、Qwen Agent 等各类 Agent 框架和 API 使用
- 04-Math：数学相关，包括连续算法、统计学、SVD 等数学知识
- 05-UT-Austin：德州大学奥斯汀分校的课程笔记和学习资料
- 06-Life：生活相关，包括日记、签证信息、实用链接等个人生活记录
- 07-internship：实习和面试准备，包括编程面试题目和 LeetCode 刷题记录
- 08-Research-utils：科研工具，包括 LaTeX 使用、研究笔记、科研图表等
- 09-Misc：其他杂项工具，包括 Hugo、PDF 处理、微信、Windows、英语学习等
