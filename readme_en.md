This repository documents all the experiences I've encountered in my life since the beginning of 2025. It's recorded here to prevent myself from making the same mistakes again, and to help others avoid the pitfalls I've encountered.

The hierarchical structure is kept to a maximum of two levels.

The current organizational structure is:
- 01-Engineer: Engineering tools including Git, Linux commands, Python, Conda, Jupyter, tmux and other development environment configurations
- 02-LLM-RL: Large Language Models and Reinforcement Learning, including LLM training, parallelization strategies, quantization techniques, Flash Attention, TRL, VERL, etc.
- 03-LLM-Agent: AI Agent tools and usage, including Claude, MCP, Ollama, Qwen Agent and various Agent frameworks and API usage
- 04-Math: Mathematics-related content, including continuous algorithms, statistics, SVD and other mathematical knowledge
- 05-UT-Austin: Course notes and learning materials from the University of Texas at Austin
- 06-Life: Life-related content, including diaries, visa information, useful links and other personal life records
- 07-internship: Internship and interview preparation, including programming interview questions and LeetCode problem-solving records
- 08-Research-utils: Research tools, including LaTeX usage, research notes, scientific charts, etc.
- 09-Misc: Other miscellaneous tools, including Hugo, PDF processing, WeChat, Windows, English learning, etc.

If you have any questions, feel free to open an issue.
