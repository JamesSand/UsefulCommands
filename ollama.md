

### run ollama

所有 ollama 能运行的模型

https://ollama.com/library 

```
ollama serve

ollama run llama3.2
```







### install ollama

没有 sudo 怎么 install ollama

跟着这里说的来

https://github.com/ollama/ollama/issues/2111 

去 release 页面上找安装包

https://github.com/ollama/ollama/releases 

下载，解压缩之后去 bin 里边，用 `./ollama` 就能运行了
