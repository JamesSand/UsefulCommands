#### 材料清单

需要填的表格

- itinarary 打印一下
- application form 打印一下

其他材料

- 预约确认单 done

- 护照原件和首页复印件 done
- 过去的申根签证复印件 done
- 美国居留证明 + 回美签证 done
- 机票预订单 done
- 酒店预订单 done
- 财产证明，过去三个月的 bank statement 这个沿用上次的就行了

- 驾照原件和复印件 done
