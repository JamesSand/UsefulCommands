
### How to talk

最好的时间是大年三十早上九点，因为零点会被信息淹没

具体话术
```
XXX 老师 / 老板，新年早上好，您是我今年第一个想送上祝福的人。回想这一年，特别感谢您在xxx 事情上对我的指导，帮我度过了难关，您的那句“xxxxx” 在这一年里一直很激励我。新的一年祝福您马年大吉！ 
```

发之前留好时间，应对可能的回复：

比如说收到回复：谢谢你，也祝你新年快乐

要**接住情绪 + 再次表态**

```
收到您的回复太开心啦

新的一年我会继续努力，把事情做的更好，不辜负您的培养和信任！
```

最后收尾的话术
```
那我就不打扰您和家人过年啦！在此祝您全家幸福，咱们节后见！
```

记住最后一条消息必须是你发的

### 送礼

有钱人要送性价比低的。这里的逻辑是性价比高的他们平时肯定已经买了，但是性价比低的他们平时不一定会买，所以送了他们会觉得很惊喜。

送老板要送稀缺性比较高的东西。比如说一些家乡的特产，一些手工艺品，或者是一些品牌故事比较特别的，和贵人有契合点的东西。

如果有小朋友的话，一定要带一些小朋友的礼物
